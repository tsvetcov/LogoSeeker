# Детекция и распознавание логотипов
## О проекте
### Задача:
В связи с проблематикой проверки публикуемых изображений на платформе Avito, можно выделить три подпроблемы:
1) Учитывая конкуренцию и риск потери пользователей в пользу других платформ, критически важно гарантировать отсутствие в публикациях на Avito внешних ссылок и логотипов конкурентов.
2) На сегменте Avito Работа зафиксированы случаи, когда мошенники использовали изображения логотипов и брендов для создания фальсифицированных объявлений, вводя пользователей в заблуждение.
3) Особое значение имеет отсев изображений с запрещённой символикой, к примеру с нацистской символикой и логотипами мошеннических организаций, чтобы обеспечить безопасность и доверие в рамках платформы.

### Цель:
Эффективно удалять неподходящий контент, предотвращать распространение вредоносной информации, улучшать пользовательский опыт и минимизировать отток на альтернативные платформы.
